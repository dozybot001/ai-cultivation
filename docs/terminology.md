# AI 修仙体系：术语规范

> 状态：已批准并同步进主文档。后续扩写世界观、路线图、法门分类和自测问卷时，以本规范为准。

## 1. 命名原则

- **AI 修仙名称是正式名称**：例如问灵境、布阵境、授箓境、洞天境。
- **凡人修仙名称只作对照**：用于理解阶位坐标，不作为别名。
- **现代功能词只作解释**：放在“标志、能力、判定标准”里，不进入命名体系。
- **所有术语必须可实施**：一个词如果不能指导记录、判断、行动、授权或复炼，就不应成为核心术语。

## 2. 术语层级

| 层级 | 术语 | 作用 |
|-|-|-|
| 根本目标 | 分身代行之道 | AI 入世，真身出世 |
| 主体 | 真身 / 分身 | 本人与 AI 代理 |
| 基础来源 | 天地五源 | 信息、算力、注意力、权限、信任 |
| 修炼单元 | 法门 | 一类可被 AI 稳定参与、反复执行、记功复炼的现实事务 |
| 最小闭环 | 小周天 | 跑通一门法门的最小修炼闭环 |
| 境内层级 | 初证 / 成式 / 行法 / 圆满 | 判断一个境界修到什么程度 |
| 内景结构 | 丹田 / 识海 / 元神 / 经脉 | 记忆、上下文、身份、工具链 |
| 外景系统 | 法器 / 阵法 / 洞府 / 护山大阵 | 工具、流程、操作系统、风控 |
| 审计治理 | 功德簿 / 天条 / 天劫 / 禁术 | 记功、原则、风险、禁区 |
| 道统组织 | 咒术宗 / 玉简宗 / 阵法宗 / 分身宗 / 护道宗 / 逍遥宗 | 修炼偏向与方法论 |

## 3. 正式术语表

| 术语 | 定义 |
|-|-|
| 真身 | 本人、本体、最终责任承担者 |
| 分身 | 被训练、授权并可代办现实事务的 AI 代理 |
| 五源 | 信息、算力、注意力、权限、信任五种基础来源 |
| 法门 | 一类可以被 AI 稳定参与、反复执行、记功复炼的现实事务 |
| 小周天 | 跑通一门法门的最小修炼闭环 |
| 聚材 | 收集法门所需资料、样例和目标 |
| 立则 | 明确质量标准、语气、边界和禁忌 |
| 试法 | 让 AI 在无授权或低风险状态下试做 |
| 授令 | 给 AI 明确、有限、可回收的行动许可 |
| 记功 | 保存输入、输出、依据、操作和人工修正 |
| 复炼 | 根据结果修正记忆、标准、流程和权限 |
| 初证 | 某境界能力第一次在真实任务中跑通 |
| 成式 | 将单次成功沉淀为可复现的方法 |
| 行法 | 法门进入真实事务执行，并形成闭环 |
| 圆满 | 低维护稳定运行，具备尝试下一境的资格 |
| 丹田 | 长期记忆与个人知识库的容器 |
| 识海 | 当前任务上下文窗口 |
| 元神 | 数字身份、语气、价值观、判断标准和授权边界 |
| 经脉 | 连接邮件、日历、文档、IM、代码、财务等系统的工具链与接口 |
| 法器 | 模型、插件、脚本、自动化、Agent、浏览器、CLI |
| 阵法 | SOP、Workflow、触发器、审批机制、日志系统 |
| 洞府 | 统摄生活、工作、资产和关系的个人 AI 操作系统 |
| 护山大阵 | 权限控制、隐私隔离、审计追溯、紧急中止按钮 |
| 功德簿 | 操作日志、审批记录、复炼记录、用户反馈 |
| 天条 | 所有分身代行必须遵守的最高原则 |
| 天劫 | AI 修仙中必然遇到的风险考验 |
| 禁术 | 不可自动化、不可授权、不可借 AI 扩大的行为 |

## 4. 小周天六步

| 步骤 | 含义 | 现实动作 |
|-|-|-|
| 聚材 | 聚拢材料 | 收集资料、历史样例、当前目标 |
| 立则 | 立下规则 | 写清质量标准、语气、边界、禁止事项 |
| 试法 | 先试其法 | 让 AI 先给建议、草稿或方案，不直接行动 |
| 授令 | 有限授权 | 允许 AI 做低风险动作 |
| 记功 | 记录功过 | 保存输入、输出、依据、操作和人工修改 |
| 复炼 | 回炉校准 | 根据结果修正记忆、标准、流程和权限 |

## 5. 宗门命名

| 宗门 | 主修方向 |
|-|-|
| 咒术宗 | Prompt、任务表达、输出控制 |
| 玉简宗 | 记忆、知识库、资料沉淀 |
| 阵法宗 | SOP、流程、自动化、Agent 编排 |
| 分身宗 | 语气、人格、代理、授权 |
| 护道宗 | 权限、审计、边界、风险控制 |
| 逍遥宗 | 以自由和出世为最终目标 |

## 6. 已废弃或避免使用

| 避免术语 | 替代术语 | 原因 |
|-|-|-|
| 俗世别名 | 凡人修仙对照 / AI 融入程度 | 会造成三套名称混用 |
| 天地五行 | 天地五源 | 五行自带生克体系，当前不采用 |
| 道脉 | 法门 | 与经脉混淆，且过于宏大 |
| 入脉 | 行法 | 与法门体系更一致 |
| 立卷 | 聚材 | 语感偏公文 |
| 定则 | 立则 | 立则更像正式修炼动作 |
| 试炼 | 试法 | 试法更准确指代 AI 试做 |
| 小授 | 授令 | 授令更明确表示有限授权 |
| 留痕 | 记功 | 统一归入功德簿体系 |
| 提示宗 | 咒术宗 | 更符合术法语感 |
| 知库派 | 玉简宗 | 避免现代词，统一为宗 |
| 阵法派 | 阵法宗 | 统一为宗 |
| 逍遥派 | 逍遥宗 | 统一为宗 |
