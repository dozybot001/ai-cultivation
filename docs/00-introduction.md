# AI 修仙体系：导读与总纲


本体系衡量的不是“用了多少 AI 工具”，而是 AI 融入本人日常生活、工作、决策、表达、关系与资产运转的程度。境界越高，AI 越能理解你的背景、遵循你的标准、使用你的工具、代表你行动。

## 0.1 红尘起念

凡人困于消息、会议、文档、任务、关系、资产与无穷选择之中。信息时代没有妖魔，妖魔就是无尽待办；没有雷劫，雷劫就是每天醒来时堆满屏幕的红点、提醒和未决事项。

于是 AI 修仙不是逃离现实，而是重建现实。修士不再幻想肉身飞升，而是炼出分身，让分身入世代掌凡务；真身从红尘杂务中赎回注意力，重新保有判断、品味、信用、关系和自由意志。

```mermaid
flowchart LR
  A[凡人用 AI] --> B[与 AI 共修]
  B --> C[AI 代行本人]
  C --> D[真身逍遥物外]
  A --> A1[效率]
  B --> B1[共识]
  C --> C1[授权]
  D --> D1[自由]
```

| 三大阶层 | 修炼目标 | 核心跃迁 |
|-|-|-|
| 下境界：筑基五境 | 把 AI 从玩具变成日常工具 | 偶尔使用 → 稳定工作流 |
| 中境界：分身四境 | 把 AI 从助手炼成分身雏形 | 辅助本人 → 主动代办 |
| 上境界：飞升六境 | 让 AI 成为现实代理 | 代办事务 → 代行人生系统 |

> AI 不替代人，AI 替人入世。凡人修效率，大能修系统，仙人修自由。

## 0.2 三层落地形态

AI 修仙的落地分为三层。当前这套世界观介绍属于道统层；它负责定名、立规、说明境界和护道边界。真正让修士开始行法的，是可执行的秘籍与 Skill。观境台则是更长期的观境层形态，需要等法门执行记录足够丰富之后再逐步实现。

| 层级 | 作用 | 当前形态 |
|-|-|-|
| 道统层 | 统一世界观、术语、境界和护道边界 | 本项目 `docs/` 文档 |
| 法门层 | 让 Agent 按道统辅助修士修炼 | `skills/ai-cultivation` 与后续可安装 Skill |
| 观境层 | 根据修炼证据诊断境界、短板和路线 | 观境台，长期展望 |

第一本修炼秘籍不是另一个长篇说明文，而是一个可安装的 Skill：让其他 Agent 读懂道统、判断境界、选择法门、带修士跑通小周天，并把稳定工作流继续炼成新的 Skill。
